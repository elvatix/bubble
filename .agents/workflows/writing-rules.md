## Text Content Rules

1. **Never use em dashes (—) or en dashes (–)** in any text content, metadata, descriptions, or code comments. Always use a regular hyphen-minus (-) with spaces around it instead.
   - ❌ `"Veilig, betrouwbaar — en zonder risico"`
   - ✅ `"Veilig, betrouwbaar - en zonder risico"`

2. **Language**: All user-facing text on this website is in **Dutch (NL)**.

3. **No emojis**: Use SVG icon components from `@/components/icons/Icons` instead of emoji characters.
